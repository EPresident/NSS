Neo Savegame Scavenger v1.003 - 21/02/2015

This utility program will help you managing your NEOScavenger savegames. It allows you to keep multiple backups of your savegames, and restore any of them at will.

Features

  • Instant backup, each backup is named by the current date and hour
  • Quick restore: restores the most recent backup available
  • Normal restore: choose a backup to restore from those available
  • Clear the savegame of NEOScavenger (deleting all the files inside the save folder)

Launching the program for the first time

The first time you launch the program you will be asked to provide the paths to your savegame and backup folders. To do so, open the Options menu ( the "Cog" button ), insert the paths, then click "Confirm & exit". A config file will be generated to store your preferences. Please note that only valid paths to existing folders will be accepted by the program. Edit the config file at your own risk.
