Neo Savegame Scavenger v1.0

This utility program will help you managing your NEOScavenger savegames. Please be aware that this piece of software comes without any kind of warranty, and I won't be responsible for any misplacement or damage or loss of data. Feel free to contact me for suggestions and bug reports.

Features

  • Instant backup, each backup is named by the current date
  • Quick restore: restores the latest backup available
  • Normal restore: choose a backup to restore from those available
  • Clear the savegame of NEOScavenger (deleting all the files inside the save folder)

Launching the program for the first time

The first time you launch the program you will be asked to provide the paths to your savegame and backup folders. To do so, open the Options menu ( the "Cog" button ), insert the paths, then click "Confirm & exit". A config file will be generated to store your preferences. Please do not tamper with the config.txt file!!!
